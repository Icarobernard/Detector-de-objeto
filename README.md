# Detector de objeto
Implementação do TensorFlow para reconhecimento de objetos 
![](https://github.com/Icarobernard/Detector-de-objeto/blob/master/src/gif1.gif)
# Requisitos
- Node.js 
# Instalação
No terminal, digite:
```md
npm install
```
Depois de baixar as dependências digite:

```md
  npm start
```
